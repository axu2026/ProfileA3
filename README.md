# Assignment 3: Profile Header
This app is for assignment 3 of CS 501 E1, creating a mock profile header using Material 3 components, modifiers, and Box composable for layout and overlapping. The app obviously lacks functionality and only serves as UI design learning.

A Box is used to create a background with a header banner, then a Card is placed on top inside the Box to overlap it, and then an Image avatar is placed inside the box to be on top of the card, creating the profile header look. Padding was the primary method used for creating the overlapping effect instead of offset or align.

## AI Disclosure
ChatGPT was used to assist in debugging, such as in the process of laying out the Card and how to implement custom images for the avatar.
